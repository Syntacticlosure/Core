# core
Scheme's tiny function


***vector->array***		
		
 `(vector->array vector)`		
		
		
```
 `#(value1 value2 value3 ...)                    =>     ((0 . value1)(1 . value2)(2 . value3) ...) 		
```		
		
***array->vector***
		
 `(vector->array list)`		
		
```		
 `((0 . value1)(1 . value2)(2 . value3) ...)     =>     #(value1 value2 value3 ...)		
```
